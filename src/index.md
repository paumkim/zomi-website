## One Big Idea

Zomi is spoken as single chunks, but often written as separate syllables.
The secret to clean Zomi writing is: **merge everything that belongs together.**

| Wrong ✗ | Right ✓ |
|---------|---------|
| `mi te` | `mite` |
| `paai ding` | `paaiding` |
| `om ta` | `omta` |
| `Zeisu in` | `Zeisu'n` |
| `ki khawm` | `kikhawm` |
